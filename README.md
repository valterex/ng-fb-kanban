## Usage

1.  Run

- `git clone`
- `cd ng-fb-kanban`
- `npm i`

2.  Update the `environment.prod.ts` and `environment.ts` files. 

```typescript
export const environment = {
  production: false,
  firebase: {
// update here
  }
};
```

3.  Run `ng serve`

Don't forget to check out the [awesome guide to this project](https://fireship.io/courses/angular/).
